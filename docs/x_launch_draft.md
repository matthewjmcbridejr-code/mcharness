# X Launch Draft

McHarness is a local-first agentic harness for supervised AI work. The backend owns workflow truth, LangGraph checkpointing, local MCP tools, and a fake-worker-only runner. The web cockpit and thin Tauri shell are both local wrappers over the same verified API surface.

What it is not: a public SaaS, an autonomous agent launcher, or production-readiness proof.
